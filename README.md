# 🔐 Secure Login System

## About
This is a beginner cybersecurity project built in Python that simulates a simple login authentication system.

## Features
- Username & password authentication
- Secure password input (hidden typing)
- Limited login attempts
- Basic account lock simulation

## Purpose
This project helps understand how authentication systems work in cybersecurity.
